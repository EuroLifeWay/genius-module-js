Introduction What is Webpack? Webpack, with the help of its loaders and plugins,
is a Module Bundler + Taskrunner.

Webpack works with most of the modern javascript module systems — AMD, CommonJS,
ES6. What this means is that javascript applications, that use any of these
module patterns, can be bundled or packagaed using Webpack.
